# Plateforme de Gestion des Stagiaires - SETIC

Une application web développée pour le SETIC 
en digitalisant la gestion des stagiaires.

##  Technologies
- **Front-end** : HTML5, CSS3, JavaScript
- **Back-end** : PHP
- **Base de données** : MySQL

##  Fonctionnalités
- [x] Inscription et authentification
- [x] Tableau de bord Admin
- [x] Suivi des stages
- [x] Donner des formations
- [x] Export PDF
